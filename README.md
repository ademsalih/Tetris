# Tetris

This is an implementation of the classical console game from the 80s. It's implemented in Java 13 and JavaFX.

### Gameplay

### Keyboard Shortcuts

| Key           | Action        |
| ------------- |:-------------:|
| N             | New Game      |
| P             | Pause         |
| Z             | Rotate back   |
